# Investigating-Netflix-Movies
This project analyzes Netflix movie data from the 1990s, focusing on identifying the most frequent movie durations, exploring genre distributions, and quantifying the number of short action movies, using Python for data manipulation and visualization.

# Project objectives
Identify the most frequent movie duration in the 1990s.
Visualize the distribution of movie genres released during the decade.
Quantify the number of short action movies (less than 90 minutes) released in the 1990s.

# Data
The dataset (netflix_df) contains information about movies available on Netflix, including columns such as duration (in minutes) and genre.

# Methods
Data Manipulation: Pandas is used to filter, aggregate, and analyze the data.
Visualization: Matplotlib or Plotly is used to visualize genre distributions.

# Key Findings
The most common movie duration in the 1990s.
The most and least popular genres during the decade.
The count of short action movies released.

# Usage
Clone this repository.
Open the notebook in your preferred Jupyter environment.
Run the cells to reproduce the analysis and visualizations.

# Requirments
Python 3.x
pandas
matplotlib or plotly
